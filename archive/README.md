# Archive

This folder stores files that were removed from the live root-site dependency graph after the audit in `docs/root-dependency-audit.md`.

These files were archived rather than deleted so they can be restored if an old route, variant, concept page, or asset becomes useful again.

Archived here:

- legacy redirect entry points
- unlinked Olea Media variant files
- unlinked Olea Tax concept pages
- unlinked runtime assets and PDFs
